# # NETmcServer

To join The Networks Minecraft server you must donwload this as a ZIP file

| --- | --- | --- |
| What you need | Where to download (Full links) | Description |
| Curse Forge | https://www.curseforge.com/download/app | Primary mod loader used |
| NETmcServer modpack | https://github.com/Gigg1E/NETmcServer | Where to donwnload modpack |
| NETmcServer Address | yin-unabashed.gl.joinmc.link | The servers address and how you join|
| The Network Discord Server | https://discord.gg/5cX7uurJEM | How to reach out and keep updated |

> The Server Address might change. Ill let you know in the discord server if it does.



## How to

1. Click the blue `Code` button to the top right of this branch

2. After the dropdown is visible, click download as zip

3. Once downloaded navigate to Curse Forge

4. Click import in Curse Forge

5. Click `choose .zip file`

6. Navigate to the location you downloaded the modpack

7. click open

Now that you have the modpack installed click play to launch



## How to add server

1. Finish editing your settings
2. Make sure you can see `Multiplayer` (Main Menu)
3. Click add server
4. Feel free to change the server name to anything you like: `NETmcServer`
5. Copy this address: `yin-unabashed.gl.joinmc.link`
6. Paste the address into the `Server Address` section
7. Click play and enjoy!
